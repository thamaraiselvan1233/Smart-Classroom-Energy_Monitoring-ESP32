# Smart Classroom Energy Monitor using ESP32

## Project Overview
This ESP32-based system detects classroom occupancy and unnecessary lighting to save energy.  
Green LED indicates normal condition, Red LED + buzzer alerts when lighting exceeds threshold.

## Components
- ESP32 Microcontroller
- PIR Sensor
- LDR
- Green LED
- Red LED
- Buzzer
- Resistors
- Wokwi Simulator

## How it works
- PIR detects motion  
- LDR measures light intensity  
- Alerts trigger if classroom is occupied and light too high
